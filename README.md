## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/rahul1905-sk/zocket

# Access
$ cd zocket

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
